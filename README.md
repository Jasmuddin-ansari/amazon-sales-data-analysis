Amazon Sales Data Analysis

-------------------------------------------------------------------------------------------------------------------------------------------------------------------
Project Overview

This project explores an Amazon sales dataset using Python.
The main purpose of the analysis is to understand sales patterns, product demand, and regional performance.

The dataset contains information about orders, product categories, sales amount, shipping locations, and order fulfilment.
By analyzing this data, we can identify trends and generate useful business insights.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------

Project Objectives

The main objectives of this analysis are:

- Understand monthly sales trends
- Identify top performing states and cities
- Analyze category-wise revenue distribution
- Study order fulfilment patterns
- Generate insights from sales data

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------

Tools Used

The following tools were used in this project:

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

These tools were used for data cleaning, analysis, and visualization.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------

Data Cleaning

Before starting the analysis, the dataset was cleaned to improve data quality.

Cleaning steps included:

- Removing unnecessary columns
- Handling missing values
- Converting the date column to datetime format
- Removing rows with missing sales values
- Standardizing column names

These steps ensure that the dataset is ready for analysis.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------

Analysis Performed

Several analyses were performed to understand the dataset.

Monthly Sales Trend

Sales were analyzed month by month to observe revenue patterns over time.

State-wise Sales

Sales were grouped by shipping state to identify the states generating the highest revenue.

Category Performance

Product categories were analyzed to see which category contributes the most to total sales.

Fulfilment Analysis

Orders were analyzed based on fulfilment type to understand the distribution between Amazon fulfilled and merchant fulfilled orders.

City-wise Sales

City level analysis was used to identify cities with the highest sales.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------

Key Insights

From the analysis, several useful insights were identified:

- Sales show variation across months indicating seasonal demand patterns.
- Some states contribute a significant share of total revenue.
- Clothing category generates the highest sales among all categories.
- Major metropolitan cities contribute the largest portion of revenue.
- Amazon fulfilled orders represent a large share of order deliveries.

Dataset
